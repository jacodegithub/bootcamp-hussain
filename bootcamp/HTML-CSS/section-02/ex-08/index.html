Open System Interconnection (OSI) Model Layers

The OSI model is a conceptual framework used to understand and describe network communication between computing systems. It consists of seven layers, each serving specific functions:

  • Physical Layer : Responsible for the physical connection between devices and the transmission of raw bits over a communication channel.
  • Data Link Layer : Handles the reliable transfer of data frames between nodes over a physical layer connection.
  • Network Layer : Manages the addressing and routing of data packets between different networks.
  • Transport Layer : Ensures the reliable and efficient delivery of data between endpoints, handling issues such as congestion control and data integrity.
  • Session Layer : Establishes, maintains, and terminates connections between applications on different network devices.
  • Presentation Layer : Handles the translation, encryption, and compression of data for effective transmission across different systems.
  • Application Layer : Provides network services to user applications and enables communication between end-user software and the network.
